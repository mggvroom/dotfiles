

# What this dotfiles collection contains

- This dotfiles collection cotains my personal configuration file which include vim.rc, .bashrc, and ansible files.
- This files allows me to setup my linux environment on a new system the same way it was on the old one.



# How to clone / utilize the dotfiles in this repo


- To clone it you'll need the ssh url from the repo and from your terminal type ``` git clone {ssh url}```.
